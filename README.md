# Telecom Churn Analysis

Predicting customer churn and recommending product strategies using Python, SQL, and Tableau.

## Project Goal
Analyze telecom customer data to identify churn drivers, predict churn risk, and recommend product actions to improve retention and revenue.

## Tools
- Python (Pandas, NumPy, scikit-learn)
- SQL (PostgreSQL/SQLite)
- Tableau
- Excel (support)
